Common causes (proxy only)

  • macro_skipped = tier_total — template only has discovery prototypes; poller cannot GET. Add scalar items (e.g. sysUpTime) or extend agent for LLD.
  • snmp_failed = all tries — wrong community / SNMPv3 / ACL / wrong IP / device off.
  • MQTT connected: no — broker URL/firewall; payloads stay in SQLite queue until connect.
  • pub > 0, keys visible, mqtt ok — agent side is fine; if nothing appears upstream, that is outside this agent (broker consumer / MDM).
